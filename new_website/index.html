AI agents are powerful — but building them is a mess

Today you stitch together Python scripts, prompt templates, and framework boilerplate. When step 12 breaks, you trace through logs, guess at context, and pray.

There's no clear way to describe what an AI agent should do step-by-step, what data each step can see, or how to debug it when things go wrong. You need a language designed for this.

Just describe what you want

Write your AI workflow in NormCode. Indentation defines data flow. Each step says what it does and what it receives — in plain language.

Document Analysis Agent
<- executive summary <= generate summary from flagged items <- discrepancy flags <= check for mismatches <- extracted figures <= extract financial data <- raw document <- database results

That's the whole agent

No framework code. No prompt chains. Just a clear description of what to do and what data to pass. The compiler handles the rest.

Indentation = data flow.
extract financial data sees only raw document.
check for mismatches sees extracted figures + database results.
generate summary sees only discrepancy flags.

Each step only sees what you explicitly pass. No hidden context leaking between steps — that's data isolation by design.

Why a language — not another framework

Frameworks give you building blocks. A language gives you a way to think. NormCode is designed from the ground up for AI agent workflows.

Human-Readable

Anyone can read a NormCode plan — not just the developer who wrote it. Share workflows with non-technical teammates.

Clean Data Flow

Each step only sees what you pass it. No context pollution, no mystery inputs. Debug any step by checking what it received.

AI Can Write It Too

Tell an LLM what you need — it generates the NormCode plan. The compiler validates and structures it automatically.

Visual Editor & Debugger

Canvas App shows your workflow as a graph. Set breakpoints, watch data flow in real time, and edit steps visually.

Pay Only for Thinking

Only reasoning steps use LLM tokens. Data routing, filtering, and looping are free. A 20-step agent might only call an LLM 8 times.

Pause, Resume, Fork

Every step is checkpointed. Pause a running agent, resume it tomorrow, or fork from any point to try a different path.
