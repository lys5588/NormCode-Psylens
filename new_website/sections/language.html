三个符号。自然语言。这就是全部语法。

<- 这是内容 数据、文档、结果——流经工作流的名词
<= 这是动作 AI执行的操作——提取、检查、生成
<* 循环 / 时序状态 重复、等待、条件分支
文档分析.ncds 
1<- 执行摘要
2    <= 根据标记项生成摘要
3    <- 差异标记
4        <= 检查不匹配项
5        <- 提取的数据
6            <= 提取财务数据
7            <- 原始文档
8        <- 数据库结果
⤡ 缩进 = 作用域 每个步骤只能看到其缩进内的数据。没有全局上下文，步骤间没有数据泄漏。
↑ 从内到外，从上到下 最深缩进先执行，再向外层推进。同一层级内从上到下执行。

其他所有内容都用自然语言编写。

这就是一个完整的AI智能体工作计划。7行诺码。从内到外、从上到下执行：从文档中提取数据 → 检查不匹配 → 汇总标记。

缩进 = 每个步骤能看到什么。提取步骤只能看到原始文档。不匹配检查看到提取的数据 + 数据库结果——仅此而已。没有全局上下文。步骤之间没有数据泄漏。
