Describe → Compile → Review → Run → Modify → Deploy

1

Describe

Write the plan — or have AI generate it from a description

2

Compile

The compiler transforms it into an executable workflow.

3

Review

Read every step before it runs. A manager can approve it.

4

Run

The orchestrator runs each step. Every input/output is recorded.

5

Modify

Change one step without breaking others. Fork from any checkpoint.

6

Deploy

Ship it. The file is yours — git it, publish it, hand it to a colleague.
