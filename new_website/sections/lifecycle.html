Describe → Review → Run → Inspect → Modify → Share

1

Describe

Write the plan — or have AI generate it from a description

2

Review

Read every step before it runs. A manager can approve it.

3

Run

The compiler transforms it. The orchestrator runs each step.

4

Inspect

Every input/output recorded. Debug any step by checking what it received.

5

Modify

Change one step without breaking others. Fork from any checkpoint.

6

Share

The file is yours. Git it, publish it, hand it to a colleague.
