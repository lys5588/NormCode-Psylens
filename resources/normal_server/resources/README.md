NormCode Canvas - Resources
============================

This folder contains resources for application packaging.

Required Files:
---------------
1. icon.ico - Application icon (Windows ICO format)
   - Recommended sizes: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256
   - Used for: executable icon, window title bar, taskbar, shortcuts

2. Psylensai_log_raw.png - Source icon image (PNG format)
   - Used as source to generate both Windows .ico and macOS .icns
   - Should be high resolution (at least 1024x1024 for best quality)
   - The build scripts will automatically convert this to platform-specific formats

Optional Files:
---------------
3. icon.icns - macOS application icon (ICNS format)
   - If not present, will be auto-generated from Psylensai_log_raw.png
   - Required sizes: 16, 32, 64, 128, 256, 512, 1024 (plus retina @2x)
   - Used for: macOS app bundle, Dock, Finder

4. dmg-background.png - DMG installer background (PNG format)
   - Recommended size: 800x600 or 1024x768
   - Used for: Custom background in DMG installer window

5. logo.png - Application logo for splash screen or about dialog

6. banner.bmp - Installer banner (for Inno Setup, 164x314 pixels)

7. wizard.bmp - Installer wizard image (for Inno Setup, 55x58 pixels)

Creating Icons:
--------------

Windows icon.ico:
You can create an ICO file using:
1. GIMP (free): File > Export As > .ico
2. IcoFX (commercial)
3. Online tools: https://icoconvert.com/
4. Auto-generated by build_windows.py from Psylensai_log_raw.png

macOS icon.icns:
You can create an ICNS file using:
1. Build script (recommended): Run build_macos.py, it will auto-generate from PNG
2. Manual: Use iconutil (requires Xcode Command Line Tools):
   ```bash
   mkdir icon.iconset
   sips -z 16 16 Psylensai_log_raw.png --out icon.iconset/icon_16x16.png
   sips -z 32 32 Psylensai_log_raw.png --out icon.iconset/icon_32x32.png
   # ... repeat for all sizes ...
   iconutil -c icns icon.iconset -o icon.icns
   ```
3. Online tools: https://cloudconvert.com/png-to-icns

Note:
- If icon.ico is not present on Windows, build will use default Python icon.
- If icon.icns is not present on macOS, build_macos.py will generate it automatically
- The Psylensai_log_raw.png is master source for both platforms

Version Management:
------------------
Version numbers are managed in build/installer.iss (Windows) and are automatically
synced to build_macos.py (macOS). To update the version:
1. Edit build/installer.iss: #define MyAppVersion "1.0.3-alpha"
2. Run build_windows.py or build_macos.py - version will be automatically detected
