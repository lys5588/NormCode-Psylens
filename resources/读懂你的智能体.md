工作流编排
Dify、Coze等。拖拽锁定的界面。需要为特定的工作方式仔细打造工作流，不适合复杂场景。


NormCode 诺码 能以计划直接约束智能体做什么，以最少形式化与自然语言结合，用任何人都能读懂、任何机器都能运行，AI可直接生成，并可以扩展、组合、复用。




三个符号。自然语言。这就是全部语法。
<-	这是内容	名词
<=	这是动作	动词
<*	循环/时序状态	条件
其他所有内容都用自然语言编写。

文档分析.ncds
<- 执行摘要
    <= 根据标记项生成摘要
    <- 差异标记
        <= 检查不匹配项
        <- 提取的数据
            <= 提取财务数据
            <- 原始文档
        <- 数据库结果
这就是一个完整的AI智能体工作计划。7行诺码。从上往下，自内到外工作。

缩进 = 每个步骤能看到什么。提取步骤只能看到原始文档。不匹配检查看到提取的数据 + 数据库结果——仅此而已。没有全局上下文。步骤之间没有数据泄漏。
